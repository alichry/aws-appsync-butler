# Installation

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

AWS AppSync Butler should be installed in your CDK or SST application.

## Create a new CDK or SST application

If you have an existing CDK or SST application setup, go to [Install dependencies](#install-dependencies).
<!--
Alternatively, you may wish to use our [starter SST template](https://example.org) to get started quickly.
-->
- Getting started with [AWS Cloud Development Kit](https://docs.aws.amazon.com/cdk/v2/guide/hello_world.html)
- Getting started with [Serverless Stack Toolkit](https://docs.serverless-stack.com/installation)

## Install dependencies

Open a terminal in your CDK or SST project and install the required dependencies.

<Tabs>
  <TabItem value="Serverless Stack Toolkit">

```shell
# Using npm:
npm install aws-appsync-butler
# Using Yarn:
yarn add aws-appsync-butler
```

  </TabItem>
  <TabItem value="AWS Cloud Development Kit">

```shell
# Using npm:
npm install aws-appsync-butler @aws-cdk/aws-appsync
# Using Yarn:
yarn add aws-appsync-butler @aws-cdk/aws-appsync
```

  </TabItem>
</Tabs>
